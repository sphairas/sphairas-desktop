Zur Wahrung des Datenschutzes wird ein Teil der personenbezogenen Daten verschlüsselt gespeichert. Dazu gehören alle Dateien, die Klarnamen von Schülerinnen und Schülern enthalten. Alle zur Verschlüsselung verwendeten Schlüssel werden in einem benutzereigenem Schlüsselbund abgelegt, der beim ersten Start des Programms erstellt werden muss. Der Schlüsselbund wird durch ein General-Passwort gesichert, das vom Benutzer festgelegt werden muss.
